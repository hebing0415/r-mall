# mall-web
小R商城 h5 后端，基于SpringBoot + JWT +Redis +MongoDb+RocketMQ实现的，主要包括首页，分类，购物车，下单，优惠券，目前商品的加购下单及微信的支付已经打通，有问题可以访问主页咨询我哦

### 项目运行
 项目环境 
>注册配置中心： Nacos(必须)

>JAVA运行环境： JDK8.0(必须)

>项目构建：maven 3.0 +(必须)

>数据库：mysql8.0,redis ,mongodb(必须)

>消息中间件: rocketMq(非必须)

> 环境都有的话，将application.yml 配置一下 , 然后运行 商品、订单、会员、home 服务即可
